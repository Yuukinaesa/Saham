This is a Streamlit app for stock data scraping, investment profit/loss calculation, and compound interest simulations using data from Yahoo Finance.

Features:

Stock Scraper
Fetch stock data (e.g., price, dividend, P/E ratio) for multiple symbols.
Calculate the number of lots based on your capital.

Profit/Loss Calculator
Calculate potential profit/loss based on buying/selling price and fees.
Include dividend calculations if desired.

Compound Interest Calculator
Simulate compound interest growth over time with an initial investment and monthly contributions.

History
View and clear history of stock data scrapes and calculations.
